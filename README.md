# IFSP News Scraper

Este é um script simples em Python para realizar o web scraping das últimas notícias publicadas no portal oficial do **Instituto Federal de São Paulo (IFSP)**.

## Funcionalidades
- Acessa a página de "Últimas Notícias" do IFSP.
- Extrai os títulos das notícias.
- Gera o link direto para cada matéria.
- Exibe a contagem total de notícias encontradas na página.

## Tecnologias Utilizadas
- **Python 3**
- **Requests**: Para realizar as requisições HTTP.
- **BeautifulSoup4**: Para analisar e extrair dados do HTML (Parsing).
